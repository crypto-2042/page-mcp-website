## 1) Verify Ownership

- Google Search Console
  - Property: `https://www.page-mcp.org`

- Bing Webmaster Tools
  - Site: `https://www.page-mcp.org`
  - Add env var in Vercel: `BING_SITE_VERIFICATION`
