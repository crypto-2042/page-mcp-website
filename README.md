# Page MCP SDK Official Website

Bilingual Next.js website skeleton for `page-mcp-sdk`.

## Scripts

- `npm install`
- `npm run dev`
- `npm run test`
- `npm run lint`
- `npm run build`

## Route Map

- `/en`, `/zh`
- `/en/docs`, `/zh/docs`
- `/en/docs/quick-start`, `/zh/docs/quick-start`
- `/en/docs/api`, `/zh/docs/api`
- `/en/docs/faq`, `/zh/docs/faq`
